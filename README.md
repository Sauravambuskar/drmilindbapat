# Dr. Milind Bapat – Urologist Website

## Project Info

This is the official website for Dr. Milind Bapat, a urologist based in India.

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd drmilindbapat

# Step 3: Install dependencies
npm i

# Step 4: Start the development server
npm run dev
```
